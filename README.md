## Installation

Step 1: Clone Repo
```sh
git clone https://github.com/briantam23/cantina-coding-challenge.git` # or clone your own fork
```

Step 2: Install `node_modules`
```sh
npm install # or yarn install
```

## Running Locally

```sh
npm start
```
