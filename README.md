## Installation

Step 1: Clone Repo
```sh
git clone https://github.com/briantam23/recursing-tree-like-ds.git` # or clone your own fork
```

Step 2: Install `node_modules`
```sh
npm install # or yarn install
```

## Running Locally

```sh
npm start
```

## Tests

Created Tests for Functions:
